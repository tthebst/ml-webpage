Transformer Model

This is an award winning English German translation model developed by facebook research. It uses attention instead of memory units and prodices state-of-the-art results. This model was released in 2018 and won the English German translation competition.

DeepSpeech

This is the deepspeech model from mozilla. It can translate speech to text. It uses memory units like LSTM's to transcribe the input. The deepspeech was revisited over time, but the initial version was released in 2015. If you notice bad translations, it is due to a skew towards their training set. This is a research model and is not made real-world usage.
