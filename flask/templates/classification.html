Resnet18

Resnet18 uses only 5 layers. Like all Resnet models this model internally skips some layers to deal with de vanishing gradient problem. This problem expacially occurs in deep networks like Resnet. This model was introduced in 2015.

Resnet50

Resnet18 uses only 50 layers. Like all Resnet models this model internally skips some layers to deal with de vanishing gradient problem. This problem expacially occurs in deep networks like Resnet. This model was introduced in 2015.

ResNeXt

ResNeXt models are pre-trained in weakly-supervised fashion on 940 million public images with 1.5K hashtags matching with 1000 ImageNet1K synsets, followed by fine-tuning on ImageNet1K dataset. This model was released in 2018 and achieves state-of-the-art accuracy.

Alexnet

AlexNet competed in the ImageNet Large Scale Visual Recognition Challenge on September 30, 2012. It achieved an error rate of 15.3%, which was 10% lower than the previous best model. One could say this is the OG computervision model.

Densenet

Dense Convolutional Network (DenseNet), connects each layer to every other layer in a feed-forward fashion. DenseNets have several compelling advantages: they alleviate the vanishing-gradient problem, strengthen feature propagation, encourage feature reuse, and substantially reduce the number of parameters. This model was released in 2016.

Googlenet

GoogLeNet was based on a deep convolutional neural network architecture codenamed “Inception”, which was responsible for setting the new state of the art for classification and detection in the ImageNet Large-Scale Visual Recognition Challenge 2014.

Mobilenet

The MobileNet v2 architecture is based on an inverted residual structure where the input and output of the residual block are thin bottleneck layers opposite to traditional residual models which use expanded representations in the input. This model was released in 2018.

Densenet

Very Deep Convolutional Networks for Large-Scale Image Recognition are models with a lot of layers. They also use small convultion filters such that trianing stays computationally possible. This model was introduced in 2014.

Squeezenet

Squeezenet achives AlexNet level results with 50x less parameters. The model size is less than 1mb and was introduced in 2017.
