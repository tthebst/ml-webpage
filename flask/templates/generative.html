Resnet18

Resnet18 uses only 5 layers. Like all Resnet models this model internally skips some layers to deal with de vanishing gradient problem. This problem expacially occurs in deep networks like Resnet. This model was introduced in 2015.

Resnet50

Resnet18 uses only 50 layers. Like all Resnet models this model internally skips some layers to deal with de vanishing gradient problem. This problem expacially occurs in deep networks like Resnet. This model was introduced in 2015.

ResNeXt

ResNeXt models are pre-trained in weakly-supervised fashion on 940 million public images with 1.5K hashtags matching with 1000 ImageNet1K synsets, followed by fine-tuning on ImageNet1K dataset. This model was released in 2018 and achieves state-of-the-art accuracy.
