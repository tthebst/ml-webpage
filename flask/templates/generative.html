Growing GANs

This model is a generative model. The purpose of this model is to generate an artifical image of a celebrity. It was trained progressivly to first generate low resolution images and at later training stages produces higher resolution images. This model was released in 2017

BigGan

This model is a generative model. The purpose of this model is to generate an artifical image of a class in the imagenet dataset. You can choose a class of a image to generate. This model was introduced in 2018.

DCGAN

This model is a generative model, which can generate people with clothes. The architecture is simple and fast, but it only generates low resolution images. It is an older model and it was released in 2015.
